# wol_bot
Навайбкодил тг-бота для включения-выключения локального сервера-файлопомойки и подсчета статистики использования траффика.
# Telegram WOL Bot (OpenWrt + OMV + Traffic Stats)

Этот бот управляет сервером (Wake-on-LAN, выключение по SSH), перезагружает роутер OpenWrt и собирает статистику интернет-трафика через conntrack.

## Возможности
- Wake-on-LAN сервера
- Выключение сервера по SSH
- Перезагрузка OpenWrt роутера
- Сбор и хранение трафика устройств до 2 лет
- Автоматическое определение новых устройств в сети
- Удобное кнопочное меню Telegram
- Автоматическое удаление старых сообщений
